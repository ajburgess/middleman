Middlenam

You're trying to connect your Angular, React or Vue web app to a remote HTTP REST API.

But the remote API has CORS protection enabled, and cannot be called by your web app.

You don't have control over the remote API, and so cannot change its CORS policy.

What can you do?

Use middleman!

Connect your web app to the middleman server, and let it make the HTTP REST API calls on your behalf. Because middleman runs server-side, it will be able to call any API, even those protected by CORS. The responses sent back from middleman include the Access-Control-Allow-Origin: * header, and so your web app API calls will always work.

If the real API URL is https://acme.com/api/add?a=1&b=2 then instead use the middleman API URL https://ajb-middleman.azurewebsites.net/https-acme.com/api/add?a=1&b=2
